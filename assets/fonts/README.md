# Font Files

Place your Amoresa font files in this directory:

- `amoresa-regular.otf` - Regular weight font file

The webpack configuration is set up to handle .otf files and will automatically include them in the build.

## Font Usage

The Amoresa font is used throughout the withFarook portfolio website with the following weights:
- 400 (regular) - for sub-brands
- 700 (bold) - for the main "withFarook" text
